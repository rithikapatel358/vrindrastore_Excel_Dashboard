# 📊 Vrinda Store Data Analysis

## 📌 Project Overview

This project focuses on analyzing sales data from **Vrinda Store** to extract meaningful insights and support business decision-making. The analysis helps understand customer behavior, sales trends, and performance across different dimensions.

---

## 🎯 Objectives

* Analyze overall sales performance
* Identify top-performing products and categories
* Understand customer demographics (age, gender, etc.)
* Track monthly and seasonal trends
* Provide actionable business insights
